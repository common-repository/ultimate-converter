== Description ==

The ultimate converter allows you to easily convert temperature, weight etc. 

* Temperature
* Length
* Weight
* Area 
* Shoe sizes
* Clothing sizes 
* Volume

Add the converter plugin to your posts/page with the shortcode [converter] or use the widget

You may install the ultimate converter plugin on your website.

Visit <a href="http://ultimateconverter.se">Ultimate Converter website</a> for more info.
